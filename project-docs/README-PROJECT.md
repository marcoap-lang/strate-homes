# Strate Homes — README-PROJECT

## Resumen ejecutivo
Strate Homes es un SaaS inmobiliario premium de la familia Strate, diseñado para agentes, equipos e inmobiliarias pequeñas o medianas que necesitan una plataforma moderna para captar leads, presentar propiedades con alto nivel visual, organizar seguimiento comercial y convertir más operaciones desde un solo sistema.

No es solo un portal inmobiliario. No es solo un CRM. No es solo un chatbot. Es una plataforma comercial integral centrada en conversión.

## Visión
Construir una plataforma SaaS escalable que combine presencia pública premium, catálogo de propiedades, detalle visual de propiedades, CRM comercial, administración multiagente, personalización controlada de presencia pública y base lista para automatización conversacional vía WhatsApp.

## Alcance actual
Primera etapa enfocada en:
- inicialización técnica del proyecto
- base del repositorio y disciplina de documentación
- arquitectura inicial del sistema
- landing premium de Strate Homes
- estructura base para panel privado
- modelo conceptual para propiedades, leads, agentes, workspaces y branding
- preparación documental para evolución a CRM, multiworkspace y bot de WhatsApp

## Stack base
- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Vercel
- Supabase (Postgres, Auth, Storage)

## Principios del sistema
- Conversión primero: cada módulo debe ayudar a captar, responder, seguir y cerrar mejor.
- Premium por defecto: diseño sobrio, aspiracional, claro y rápido.
- Mobile first: experiencia impecable en móvil antes que escritorio.
- Personalización controlada: bloques, layouts y settings estructurados, no page builder caótico.
- Multiworkspace desde la base: preparado para agente individual, equipo e inmobiliaria.
- Escalabilidad sin sobreingeniería: base sólida hoy, expansión limpia mañana.
- Continuidad persistente: decisiones, estado y próximos pasos siempre documentados.
- Mantenibilidad: menos piezas, mejor pensadas.
