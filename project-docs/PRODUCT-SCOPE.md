# Strate Homes — PRODUCT-SCOPE

## Qué sí entra
### Núcleo de primera etapa
- branding inicial de Strate Homes
- landing premium
- estructura técnica profesional
- documentación persistente robusta
- base de dominio para propiedades, leads, agentes, workspaces y branding
- shell inicial de administración
- preparación conceptual para listados y detalle de propiedad
- preparación conceptual para páginas públicas de agente y empresa
- base conceptual para flujo comercial y conversión

### Módulos preparados desde la arquitectura
- landing pública
- listado público de propiedades
- detalle de propiedad
- área privada de administración
- gestión de propiedades
- gestión de leads
- gestión de agentes y workspaces
- base de integración futura con bot de WhatsApp
- módulo de personalización de presencia pública
- base de analítica comercial futura

## Qué no entra por ahora
- bot de WhatsApp completo
- automatizaciones avanzadas
- CRM completo con todas las transiciones y reportes
- page builder libre
- marketplace o portal masivo de terceros
- sistema financiero/contable
- motor avanzado de comisiones
- white-label total con dominios y temas ilimitados

## Qué dejamos preparado pero no implementado aún
- multiworkspace real con permisos finos
- páginas públicas de empresa y agente completas
- modelo robusto de pipeline comercial
- tracking analítico profundo
- event log comercial
- desarrollos y unidades dentro de desarrollo
- pre-sales y inventario por desarrollo
- handoff automatizado bot → humano
- tiempos de respuesta y SLA operativos

## Restricciones deliberadas
- se evita complejidad prematura
- se evita libertad visual total que rompa consistencia
- se evita acoplar demasiado pronto todo a backend real
- se privilegia claridad de dominio sobre cantidad de módulos
